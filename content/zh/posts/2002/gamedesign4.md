## 游戏策划是这样练出来的（序）

在网上看久了，感觉到很多教程都是为程序员设计的，而只有少数的，将自己的策划案，或是策划总结拿出来供新手策划学习，更有不少的程序员对策划这个职务重视不够，到真正编程序的时候，才发现自己老是差了这里缺了那里，其实，策划是十分重要的，可以说，他就是整个游戏的灵魂，整个游戏的好坏，大部分就取决于他了，所以这些玩家一旦发现游戏的不好之处，准备破口大骂时，首当其冲的就是游戏策划，悲哀呀……555555……

除了重视不够之外，游戏策划的另一大悲哀就是，许多人（游戏狂人、初级玩家，甚至是不懂游戏的人）认为策划极端的好当，不过像编辑一样，拿笔写写字而已，假如他们仅是这样想那就好了，但他们偏不，常常对着一些策划大骂，仅此，中国都不知道失去了多少的优秀策划，悲哀呀……555555……

所以策划不但要了解与游戏相关的（编程、美工等）和不完全相关的（地理、天文、神话、兵器等），还要懂得如何管理团队和带动大家，但我认为最关键的就是要有非凡的忍耐力和广阔的胸襟，为的是什么？就是在“超级玩家”的猛烈攻击下能经得住考验！

其实这篇文章早就想写了，但鉴于我的策划技术，属于菜鸟级别，只怕我在第一轮攻击中就就地身亡了，所以呀……呵呵，拖到了现在，不过现在要开始写了，但几个月后的我还是一个菜鸟，那几个月应付考试去了，所以呀……呵呵，我知道的，我告诉您，您知道的，也请告诉我！

我相信一点，菜鸟教菜鸟，应该会快些，更容易懂些！

我说下我的教学安排，在策划教学中，我们将完整的制作一部策划案，其实这部策划案，我已经写了一小半了，但现在，我还是决定完全重写，一来可以让大家了解一部策划案制作的全过程，二来也可以让这个策划案更加的成熟，三来呀还可以让我从菜鸟向飞龙过渡，四来呀……（你有完没完？）

好了，废话不讲了，只要记住我们策划要义无反顾，躲过一切进攻，勇往直前！

奥！我差点忘了，这部策划案的名字叫：《冰河魔法传》，有点像我的网名对吧？对，我的网名就出自这里！哈哈哈……

## 游戏扫盲知识（一）

游戏的基本类型

- ACT......(ACTION GAME )动作游戏
- STG......(SHOTING GAME )射击游戏
- RPG......(ROLE PLAYING GAME )角色扮演游戏
- A.RPG....(ACTION ROLE PLAYING GAME )动作角色扮演游戏
- S.RPG....(SIMULATION ROLE PLAYIG GAME)模拟角色扮演游戏
- FTG......(FIGHTING GAME )格斗游戏
- S.FTG....(SIMULATION FIGHTING GAME )模拟格斗游戏
- SLG......(SIMULATION GAME )模拟仿真游戏
- SPG......(SPORT GAME )运动游戏
- TAB......(TABLE GAME )桌上游戏
- PUZ......(PUZZLE GAME )益智游戏
- AVG......(ADVENTURE GAME )冒险游戏
- RAC......(RACE GAME )赛车游戏
- RTG......(REAL TIME GAME)实时战略游戏
- PET......(PET)养成类游戏及电子宠物
- MAG......(MANAGEMENT GAME)经营类游戏
- L.MUD....(LETTER MULTI-USER DUNGEONS)文字网络游戏
- F.MUD....(FIGURE MULTI-USER DUNGEONS)图形网络游戏
- ETC......(ETCTERA GAME )其他类游戏

## 超级酷点——建议式行动

在我们的RPG游戏里，常常会有几个NPC跟着主角一起并肩作战，无论是那几个NPC一开始就喜欢主角，还是将会喜欢，她们（偶尔是“他们”，呵呵……）总会一开始就听从主角的，这显然有些不太真实，其实把这点做真实一点是非常简单的，呵呵……听我慢慢道来！

《王权》这个游戏不知你玩过没有，可以说，这是一个失败的游戏，但它的创意我们可以借鉴，在《王权》里，你无法直接控制你的部队，想让他们给你“办事”——杀敌，你就得给他们Money。中国人自古以来就重情讲义气，当然不能用金钱来让她给你办事，这里只要设置一个感情指数，当感情指数比较高时，你对NPC的操作，成功率就会比较高！（请你放心，RPG里的Girl NPC对主角总是一相情愿，它们之间的感情指数当然就别提了——其实那时叫做“恋爱指数”也不为过，呵呵……

另外，一个大点，当她（偶尔是“他”，呵呵……）身亡后，根据感情指数，可以将主角的攻击力、敏捷（但注意：绝没有防御）适当的提高。在他们全部阵亡以后，还可以根据主角与各NPC之间的感情指数以及敌人的冷酷指数，来确定主角们是否能逃过一劫。当然，还有一种，那就是当主角死后，他/她的GF/BF来对他/她做了个动作（通常都是吻，好单调呀，其实踢他/她两下、泼泼冷水也是不错呀，别弄死了就好，呵呵……）他/她就复活了，并且他们之间的感情指数大增，呵呵……

最后，如果真的死了，不可救药了，那千万别显示一个“GAME OVER”，那太没创意了，显示一个“Welcome to the fantasy heven!”都会好些！如果能设计一幅天堂的地图，旁边站一个老头（应该是一个仙人吧！）他就告诉你，你为什么不可救药了，是因为与NPC之间关系搞得不好吗，还是遇到敌人时，不懂得拍马屁……哈哈哈……越说越离题了，不说了。
