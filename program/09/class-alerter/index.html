下载地址:alerter.rar

软件介绍

是不是觉得考研很没效率？那就试试这款软件吧，安排好考研的计划，然后好好的按照提醒器进行执行吧。

具有定时打铃、统计学习时间、考验倒计时功能。 忘说了，为了方便在上课时间禁用了屏幕保护。

安装说明：

  1. 本软件属于绿色软件，直接解压至任何地方均可运行。
  2. 本软件需要.Net

需要.Net 2.0

about
